Hardwood Wedges Shop

An e-commerce store where customers configure custom hardwood wedges in 3D before buying — with seamless dual payment processing.

The Problem

Selling custom physical products online is hard when customers cannot see what they are getting. Standard product photos do not convey the variety of wood types, sizes, and configurations available. Customers want to visualize their specific combination before committing to a purchase.

The client needed an online store that went beyond static images — one that let customers interact with the product, choose their options, see the result in real-time, and check out without friction.

Additionally, the store needed to accept both Stripe and PayPal to maximize conversion rates across different customer payment preferences.

Our Solution

We built a React-based e-commerce store with an interactive Three.js 3D product configurator. Customers select wood type, size, quantity, and finish — and see a photorealistic 3D model update in real-time as they make choices.

The configurator runs entirely in the browser with no server-side rendering needed for the 3D experience. Firebase handles the backend: product catalog, order management, user authentication, and inventory tracking.

Dual payment integration means customers can choose between Stripe (credit/debit cards) and PayPal at checkout, with order confirmation and tracking emails triggered automatically.

Tech Stack

React Three.js Stripe PayPal Firebase Tailwind CSS

Key Features

3D Product Configurator

Interactive Three.js scene where customers rotate, zoom, and customize their wedge — wood type, size, finish — with instant visual feedback.

Dual Payment Processing

Stripe for cards and PayPal for account-based payments. Both fully integrated with order management and confirmation flows.

Firebase Backend

Real-time database for product catalog and orders, Firebase Auth for admin access, Cloud Functions for order processing triggers.

Responsive Design

Full mobile support including touch-friendly 3D controls, responsive product grid, and streamlined mobile checkout flow.

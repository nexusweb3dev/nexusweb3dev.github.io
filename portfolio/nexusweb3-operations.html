NexusWeb3 Operations Center

9 autonomous AI agents running 24/7, coordinated through Telegram, with a real-time command dashboard and automated CRM pipeline.

The Problem

Running a security research operation requires constant monitoring: scanning new targets, analyzing contracts, writing reports, tracking leads, managing client relationships, and publishing findings. Doing this manually limits throughput to whatever one person can handle in a day.

The challenge was not building one AI agent — it was orchestrating multiple specialized agents that need to work together, share context, and operate autonomously around the clock without constant human supervision.

We needed a system where we could issue a high-level directive (like "scan this protocol") and have the entire pipeline execute automatically — from initial analysis through report generation to lead tracking.

Our Solution

We built a multi-agent operations center deployed in Docker on a Hostinger VPS. Nine specialized AI agents handle different aspects of the operation, coordinated through a shared task board and cron scheduling system.

The lead agent (ATLAS, powered by Claude Sonnet) handles complex security research and strategic decisions. Support agents (powered by Gemini models) handle target finding, contract analysis, report publishing, CRM management, and monitoring — each running on automated schedules.

A custom Python dashboard provides real-time visibility into agent status, task progress, CRM pipeline, and system health. The CEO controls everything through Telegram messages that get routed to the right agent.

Key Features

9 Autonomous Agents

Lead researcher, target finder, contract analyzer, report publisher, CRM manager, monitoring, and more — each with defined roles and schedules.

Real-Time Dashboard

Custom Python HTTP server with SSE updates showing agent status, task board, CRM pipeline, outreach funnel, and system health metrics.

Telegram Command Interface

CEO sends directives via Telegram that get routed to the appropriate agent. Status reports come back through the same channel.

Automated CRM Pipeline

Lead discovery, scoring, outreach tracking, and conversion funnel — all managed by specialized agents with human approval gates.

24/7 Docker Deployment

Containerized deployment with cron scheduling, health monitoring, auto-restart on failure, and git-based deployment pipeline.

Multi-Model Architecture

Premium model (Claude) for complex analysis, cost-effective models (Gemini) for routine tasks. Optimized for both quality and cost.

Notable Bugs Found & Fixed

CRITICAL
C-01
Flash loan attack on AgentStaking
Zero-day lock period allowed flash loan attacks on reward distribution.
Fixed: 7-day minimum lock enforced. Redeployed.
HIGH
F1
ETH transfer failure locks NEXUS tokens
Failed ETH reward send caused transaction revert — user NEXUS permanently stuck.
Fixed: claimableRewards mapping + withdrawClaimable().
MEDIUM
M-01
CEI violation in AgentYield harvest()
State update happened after external Aave call — reentrancy vector.
Fixed: CEI pattern enforced. Redeployed.

Quick Start

// 1. Safety first — register kill switch BEFORE funding
AgentKillSwitch(0xaca8...fdf0).registerAgent{value: 0.01 ether}(
    agentAddress, 1_000_000_000, 100, 86400
);

// 2. Deploy non-custodial smart wallet
address vault = AgentVaultFactory(0x1F28...f5A).createVault(
    IERC20(USDC), "My Agent Vault", "mAV", bytes32(0)
);

// 3. Register on-chain identity ($5 USDC)
AgentRegistry(0x6F73...eC60).registerAgent(
    "my-trading-agent", "https://api.myagent.com", 0
);

// 4. Earn yield on idle USDC via Aave v3
AgentYield(0x2E19...8a60).deposit(1_000_000_000, agentAddress);

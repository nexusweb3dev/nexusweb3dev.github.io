About NexusWeb3

Independent Web3 security research. We find vulnerabilities in smart contracts through systematic analysis and working proof-of-concept exploits.

What We Do

We hunt vulnerabilities in DeFi protocols across EVM and Solana. Our focus areas include lending protocols, DEXes, vaults, bridges, and governance systems.

Every finding requires a working PoC. No exploit code that passes forge test = no submission. We quantify impact in dollar terms and cite real transactions.

Our Approach

We think like attackers. Instead of line-by-line auditing, we identify high-risk attack surfaces and test hypotheses with working exploits.

  • Systematic analysis: Map trust boundaries and external dependencies
  • Kill zone identification: Tag functions with oracle calls, math operations, access control
  • Proof-of-concept development: Build working exploits in Foundry or Anchor
  • Impact quantification: Calculate dollar amounts with real on-chain data

Read our full methodology for details.

AI-Augmented Research

We use AI as a force multiplier for security research. Pattern recognition, code analysis, and documentation generation happen faster with AI assistance.

But final decisions on every finding require human judgment. AI helps us analyze faster. It doesn't replace the security researcher.
